## GET (metoda)

### /all-charactes/\<text\>

Endpoint zwraca informację o ilości wystąpięń wszystkich obłsugiwanych znaków.

<b>Przykładowa odpowiedź<b>:
```json
{
  "big-letters": 0,
  "small-letters": 0,
  "numbers": 0,
  "special-characters": 0
}
```

### /big-letters/\<text\>

Endpoint zwraca informację o ilości wystąpięń wielkich liter.

<b>Przykładowa odpowiedź<b>:
```json
{
  "big-letters": 0,
}
```
